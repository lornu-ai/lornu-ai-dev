# Assets Directory

This directory contains brand assets for the LornuAI application.

## Logo

**Required:** Download the official logo from the Google Drive link provided in [Issue #10](https://github.com/lornu-ai/lornu-ai/issues/10) and place it here.

### Expected Files:
- `logo.png` - PNG format (optimized for web use)

### Logo Placement:
1. Download from: https://drive.google.com/file/d/1VwCBEDtaA-Rab1REXJY7Ialp8dQXkyMr/view?usp=sharing
2. Save as `logo.png` in this directory
3. Ensure proper optimization for web use

### Usage:
- Header navigation: Uses `logo.png`
- Favicon: Generated from logo (see `/public` directory)
